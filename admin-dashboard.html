📖 How the Claude API Works — From Scratch
1

🤔 What even is an API?

Think of an API like a waiter in a restaurant. You (your website) tell the waiter (API) what you want. The waiter goes to the kitchen (Claude's AI brain) and brings back the food (the AI's answer). You don't need to know how the kitchen works — you just ask and get a reply.

2

🔑 The API Key — Your Password

To use Claude's API, you need a key — like a password. It looks like: sk-ant-api03-xxxx...

Get yours free at console.anthropic.com → Sign up → API Keys → Create Key.

⚠️ Never share this key publicly! Anyone with it can use your account and spend your money.

3

💸 What Does It Cost?

Anthropic charges per message, very cheaply. Roughly:
• 1,000 messages = about ₹1–3
• A busy chatbot answering 100 messages/day = ~₹10–30/month

So if you charge clients ₹299/month and your API cost is ₹30, you keep ₹269 profit per client. At 20 clients = ~₹5,380/month.

4

📦 How a Request Works

Your website sends a request to Anthropic like this — you give it a system prompt (tells the AI who it is and what to do), a message (what the user typed), and your API key. Claude reads everything and sends back a reply in milliseconds.

5

🔒 Keeping Your Key Safe

For selling to clients, you have two options:

Easy way: Give each client their own API key. They pay Anthropic directly. You only charge for setup/customization.

Business way: Use one key on a backend server you control. Clients never see it. You charge a monthly fee that covers your API costs + profit.

6

🚀 Your Workflow as a Seller

1. Client fills your order form
2. You take their business info and fill the widget config (takes 10 mins)
3. You paste the widget code into their website
4. Client's visitors now chat with their AI
5. You collect monthly payment 💰
